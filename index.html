Resumen

Este sitio explora el conflicto Irán-Israel de 2025 a través de elementos multimedia creados con software libre en Linux.

Acerca de

Este sitio fue creado con software libre (Audacity, GIMP, Blender, Kdenlive, FFmpeg) en Linux, siguiendo los principios de programación multimedia del plan de estudios de la UNAM. Las imágenes y animaciones fueron procesadas con Python y JavaScript (Three.js).
